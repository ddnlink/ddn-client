<h1 align="center">DDN Wallet</h1>

DDN企业级区块链钱包，操作和使用 DDN 企业级区块链公链的工具。最新版本与测试网保持同步，[测试网钱包](https://wallet.testnet.ddn.net). 参与贡献并获取捐赠，请访问[主网钱包](https://wallet.ddn.net)


有关 DDN企业级区块链 更多信息，请访问官方网站：<https://ddn.net>

## 功能

- [x] 首页
- [x] 转账
- [x] 投票
- [x] 资产
- [x] 用户系统
  - [x] 登录
  - [x] 创建新账户
- [x] 区块链浏览器
- [x] 多重签名
- [x] Dapp应用市场
- [ ] 智能合约
- [ ] 签名

## 开发

本项目由 [UMI企业级前端研发框架](https://umijs.org), [ant design](https://ant.design), 以及 [DDN 企业级区块链](https://ddn.net)驱动，请自行学习相关文档和接口。

开发过程中，您可能会用到 [DDN 区块链云平台](https://ui.ddn.net)，用于设置、打包和部署新链，下载安装测试链等。

如果您是初学者，请学习我们提供的课程 [DDN-UBL 高校区块链实验课程](https://ubl.ddn.net), 该课程是针对大三、研究生等在校学生，以及在职多年希望转战区块链的小伙伴准备的，详细讲解了基于`DDN企业级区块链`的应用开发流程，提供了众多案例，对学习前端、区块链等应用开发非常有帮助。

从源码下载和启动本项目

```bash
$ git clone https://github.com/ddnlink/ddn-wallet.git
$ cd ddn-wallet
$ yarn # or npm install
$ yarn start # or npm start         
# 访问 http://localhost:8000
```

注：启动之前，应该事先安装并运行 `DDN企业级区块链`本地节点, 请参考文档[DDN 测试网（testnet）节点安装](http://docs.ddn.net/guide/peer-install-testnet)
