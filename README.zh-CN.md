<h1 align="center">DDN Wallet</h1>


## 功能

```
- 首页
- 转账
- 多重签名
- 投票
- 资产
- 区块链浏览器
- 用户系统
  - 登录
  - 创建新账户
```

## 使用

```bash
$ git clone 
$ cd ddn-wallet
$ npm install
$ npm start         # 访问 http://localhost:8000
```
