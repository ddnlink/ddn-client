<h1 align="center">DDN Wallet</h1>

## Features

```
- Home
- Transfer
- MultiSignature
- Vote
- Assets
- Browser
- User
  - Login
  - Create Account
```

## Usage

```bash
$ git clone
$ cd ddn-wallet
$ npm install
$ npm start         # visit http://localhost:8000
```
